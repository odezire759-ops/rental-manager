# J.Residence — Rental Manager

ระบบจัดการค่าเช่า เชื่อมต่อ Supabase

## Deploy บน GitHub Pages

1. สร้าง repo ใหม่ชื่อ `rental-manager` (หรือชื่ออื่นก็ได้)
2. upload ไฟล์ทั้งหมดใน folder นี้
3. ไปที่ Settings → Pages → Source: `main` branch → `/root` → Save
4. เข้าใช้งานที่ `https://odezire759-ops.github.io/rental-manager/`
